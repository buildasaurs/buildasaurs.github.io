Analytics

  • ekgclient is a tiny Swift SDK for sending heartbeats to ekg.
  • ekg-swift is the currently-used Swift implementation of the analytics server, runs on Heroku at https://builda-ekg.herokuapp.com.
  • ekg is a depricated, Node.js version of ekg.

Utils

  • xcskarel is a Ruby command line utility for managing Xcode Server.
  • satellite is a Node.js server providing build badges for Buildasaur.
  • XcodeServer-API-Docs is depricated documentation of Xcode Server API.
